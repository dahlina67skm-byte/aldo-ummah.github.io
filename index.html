Apa itu Cybersecurity?

Cybersecurity adalah praktik melindungi sistem komputer, jaringan, program, dan data dari serangan digital. Di era digital saat ini, cybersecurity sangat penting karena hampir semua aspek kehidupan kita terhubung dengan internet.

CIA Triad - Tiga Pilar Keamanan

1. Confidentiality (Kerahasiaan): Hanya pihak yang berwenang dapat mengakses data.

2. Integrity (Integritas): Data tidak diubah tanpa otorisasi.

3. Availability (Ketersediaan): Sistem dapat diakses saat dibutuhkan.

Defense in Depth (Pertahanan Berlapis)

Keamanan yang efektif memerlukan multiple layers. Jika satu layer terjadi breach, layer lain masih melindungi:

  1. Perimeter Security: Firewall, DDoS protection
    2. \n
  2. Network Security: IDS/IPS, VPN
  3. Host Security: Antivirus, Endpoint protection
  4. Application Security: Secure coding
  5. Data Security: Encryption, Backup

1. Malware (Malicious Software)

Software berbahaya yang merusak atau mengakses sistem tanpa izin. Meliputi:

2. Phishing

Teknik penipuan untuk mendapatkan informasi sensitif dengan menyamar sebagai entitas terpercaya. Menyebabkan 90% data breaches.

Tanda-tanda Email Phishing:

  • URL mencurigakan
  • Urgency tinggi ("Akun akan dibekukan!")
  • Permintaan data pribadi
  • Grammar/spelling mistakes

3. DDoS (Distributed Denial of Service)

Serangan yang membanjiri server dengan traffic berlebihan hingga tidak dapat diakses oleh pengguna legitimate.

Dampak: Layanan tidak tersedia, kerugian revenue, damage reputasi.

4. Man-in-the-Middle (MITM) Attack

Penyerang mencegat komunikasi antara dua pihak untuk mencuri atau memanipulasi data. Sering terjadi di WiFi publik.

5. Injection Attacks

Menyisipkan kode berbahaya ke dalam input aplikasi:

  • SQL Injection: Menyisipkan SQL commands
  • XSS (Cross-Site Scripting): Inject JavaScript

  • ‚úì Aktifkan automatic updates
  • ‚úì Update sistem operasi, browser, aplikasi
  • ‚úì Jangan gunakan software bajakan

3. Two-Factor Authentication (2FA)

Menambahkan lapisan keamanan ekstra dengan memerlukan dua bentuk verifikasi. Mengurangi risiko unauthorized access hingga 99.9%.

4. Firewall & Antivirus

Firewall: Memfilter traffic jaringan masuk dan keluar.

Antivirus: Mendeteksi dan menghapus malware.

5. Backup Data Reguler

Aturan 3-2-1: 3 salinan, 2 media berbeda, 1 lokasi offsite.

6. VPN untuk WiFi Publik

Gunakan VPN di WiFi publik untuk mengenkripsi traffic dan melindungi privasi.
